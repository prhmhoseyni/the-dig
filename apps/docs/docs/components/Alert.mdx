# Alert

## Preview

import Alert from "@repo/ui/Alert";
import Container from "./Container";

<Container>
  <Alert
    title="عنوان پیام"
    description="توضیحات مختصر درمورد پیام هشدار."
    className="w-full"
    onClose={() => null}
  />
</Container>

```tsx
import Alert from "@repo/ui/Alert";

<Alert
  title="عنوان پیام"
  description="توضیحات مختصر درمورد پیام هشدار."
  onClose={() => setOpen(false)}
/>;
```

## Installation

```bash
npx the-dig@latest add Alert
```

## Props

| Prop                                                               | Type                                                             | Default  |
| ------------------------------------------------------------------ | ---------------------------------------------------------------- | -------- |
| `title`                                                            | `ReactNode`                                                      | -        |
| `description` <sup className="text-red-600 text-caption2">\*</sup> | `ReactNode`                                                      | -        |
| `color`                                                            | `"gray"` \| `"info"` \| `"success"` \| `"warning"` \| `"danger"` | `"gray"` |
| `onClose`                                                          | `VoidFunction`                                                   | -        |
| `className`                                                        | `string`                                                         | -        |
