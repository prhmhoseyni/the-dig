## Props

### `Tabs` component

| Prop       | Type                         | Default |
| ---------- | ---------------------------- | ------- |
| `value`    | `number`                     | -       |
| `onChange` | `(newValue: number) => void` | -       |
| `children` | `ReactNode`                  | -       |

### `Tabs.Item` component

| Prop        | Type         | Default |
| ----------- | ------------ | ------- |
| `active`    | `boolean`    | `false` |
| `onClick`   | `() => void` | -       |
| `className` | `string`     | -       |
| `children`  | `ReactNode`  | -       |
