## Installation

```bash
npx the-dig@latest add Stepper
```

## Props

| Prop                                                         | Type                                             | Default |
| ------------------------------------------------------------ | ------------------------------------------------ | ------- |
| `steps` <sup className="text-red-600 text-caption2">\*</sup> | `Array<{ title: string; description?: string }>` | -       |
| `step` <sup className="text-red-600 text-caption2">\*</sup>  | `number`                                         | -       |
| `onChange`                                                   | `(step: number) => void`                         | -       |
| `vertical`                                                   | `boolean`                                        | `false` |
