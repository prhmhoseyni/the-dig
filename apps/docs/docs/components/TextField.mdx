## Props

| Prop                | Type                                    | Default                                                                                                        |
| ------------------- | --------------------------------------- | -------------------------------------------------------------------------------------------------------------- |
| `variant`           | `"primary"` \| `"secondary"`            | `"primary"`                                                                                                    |
| `hasError`          | `boolean`                               | `false`                                                                                                        |
| `startAdornment`    | `ReactNode`                             | -                                                                                                              |
| `endAdornment`      | `ReactNode`                             | -                                                                                                              |
| `size`                                                              | `"xs"` \| `"sm"` \| `"md"` \| `"lg"`                                              | md                                                                                                        |
| `inputWrapperProps` | `HTMLAttributes<HTMLDivElement>`        | <RedirectLink href="https://developer.mozilla.org/en-US/docs/Web/API/HTMLElement">Document</RedirectLink>      |
| Others              | `InputHTMLAttributes<HTMLInputElement>` | <RedirectLink href="https://developer.mozilla.org/en-US/docs/Web/API/HTMLInputElement">Document</RedirectLink> |
