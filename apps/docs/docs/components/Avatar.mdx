# Avatar

## Installation

```bash
npx the-dig@latest add Avatar
```

import Container from "./Container";

## Code

```tsx

```

## Props

| Prop        | Type                                 | Default     |
| ----------- | ------------------------------------ | ----------- |
| `src`       | `string`                             | -           |
| `alt`       | `string`                             | -           |
| `size`      | `"xs"` \| `"sm"` \| `"md"` \| `"lg"` | `"md"`      |
| `className` | `string`                             | -           |
| `status`    | `"online"` \| `"offline"`            | `"offline"` |
| `children`  | `ReactNode`                          | -           |
