# Button

import RedirectLink from "../../src/components/RedirectLink";

## Installation

```bash
npx the-dig@latest add Button
```

## Props

| Prop        | Type                                                                          | Default                                                                                                         |
| ----------- | ----------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------- |
| `variant`   | `"contained"` \| `"tinted"` \| `"outlined"`                                   | `"contained"`                                                                                                   |
| `color`     | `"brand"` \| `"info"` \| `"success"` \| `"warning"` \| `"danger"` \| `"gray"` | `"brand"`                                                                                                       |
| `size`      | `"xs"` \| `"sm"` \| `"md"` \| `"lg"` \| `"xl"`                                | `"md"`                                                                                                          |
| `isLoading` | `boolean`                                                                     | `false`                                                                                                         |
| Others      | `ButtonHTMLAttributes<HTMLButtonElement>` (e.g. `onClick`, `disabled`, etc.)  | <RedirectLink href="https://developer.mozilla.org/en-US/docs/Web/API/HTMLButtonElement">Document</RedirectLink> |
