# Button

## Preview

import Button from "@repo/ui/Button";
import Container from "./Container";

<Container>
  <Button>فشارم بده</Button>
</Container>

## Code

```tsx
import Button from "@repo/ui/Button";

<Button variant="contained" color="brand">
  فشارم بده
</Button>;
```

## Installation

```bash
npx the-dig@latest add Button
```

## Props

| Prop        | Type                                                                          | Default                                                                        |
| ----------- | ----------------------------------------------------------------------------- | ------------------------------------------------------------------------------ |
| `variant`   | `"contained"` \| `"tinted"` \| `"outlined"`                                   | `"contained"`                                                                  |
| `color`     | `"brand"` \| `"info"` \| `"success"` \| `"warning"` \| `"danger"` \| `"gray"` | `"brand"`                                                                      |
| `size`      | `"xs"` \| `"sm"` \| `"md"` \| `"lg"` \| `"xl"`                                | `"md"`                                                                         |
| `isLoading` | `boolean`                                                                     | `false`                                                                        |
| `...props`  | `ButtonHTMLAttributes<HTMLButtonElement>` (e.g. `onClick`, `disabled`, etc.)  | [Document](https://developer.mozilla.org/en-US/docs/Web/API/HTMLButtonElement) |
