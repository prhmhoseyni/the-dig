# IconButton

## Preview

import IconButton from "@repo/ui/IconButton";
import Container from "./Container";
import { Activity } from "lucide-react";

<Container>
  <IconButton>
    <Activity size={18} />
  </IconButton>
</Container>

## Code

```tsx
<IconButton variant="contained" color="brand">
  <Activity size={18} />
</IconButton>
```

## Installation

```bash
npx the-dig@latest add IconButton
```

## Props

| Prop        | Type                                                                                 | Default                                                                            |
| ----------- | ------------------------------------------------------------------------------------ | ---------------------------------------------------------------------------------- |
| `variant`   | `"contained"` \| `"tinted"` \| `"outlined"`                                          | `"contained"`                                                                      |
| `color`     | `"brand"` \| `"info"` \| `"success"` \| `"warning"` \| `"danger"` \| `"gray"`        | `"brand"`                                                                          |
| `size`      | `"xs"` \| `"sm"` \| `"md"` \| `"lg"` \| `"xl"`                                       | `"md"`                                                                             |
| `isLoading` | `boolean`                                                                            | `false`                                                                            |
| `...props`  | `IconButtonHTMLAttributes<HTMLIconButtonElement>` (e.g. `onClick`, `disabled`, etc.) | [Document](https://developer.mozilla.org/en-US/docs/Web/API/HTMLIconButtonElement) |
