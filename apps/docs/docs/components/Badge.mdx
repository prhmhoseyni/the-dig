# Badge

## Preview

import Badge from "@repo/ui/Badge";
import Container from "./Container";

<Container>
  <Badge>به من توجه کن</Badge>
</Container>

## Code

```tsx
<Badge variant="contained" color="brand">
  به من توجه کن
</Badge>
```

## Installation

```bash
npx the-dig@latest add Badge
```

## API

| Prop       | Type                                                                          | Default                                                                  |
| ---------- | ----------------------------------------------------------------------------- | ------------------------------------------------------------------------ |
| `variant`  | `"contained"` \| `"tinted"`                                                   | `"contained"`                                                            |
| `color`    | `"brand"` \| `"info"` \| `"success"` \| `"warning"` \| `"danger"` \| `"gray"` | `"brand"`                                                                |
| `...props` | `HTMLAttributes<HTMLDivElement>` (e.g. `onClick`, `disabled`, etc.)           | [Document](https://developer.mozilla.org/en-US/docs/Web/API/HTMLElement) |
