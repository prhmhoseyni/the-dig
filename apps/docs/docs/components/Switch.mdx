## Props

| Prop    | Type                                                                          | Default                                                                                                        |
| ------- | ----------------------------------------------------------------------------- | -------------------------------------------------------------------------------------------------------------- |
| `color` | `"brand"` \| `"info"` \| `"success"` \| `"warning"` \| `"danger"` \| `"gray"` | `"brand"`                                                                                                      |
| `size`  | `"sm"` \| `"md"` \| `"lg"`                                                    | `"md"`                                                                                                         |
| Others  | `InputHTMLAttributes<HTMLInputElement>` (e.g. `onChange`, `disabled`, etc.)   | <RedirectLink href="https://developer.mozilla.org/en-US/docs/Web/API/HTMLInputElement">Document</RedirectLink> |
