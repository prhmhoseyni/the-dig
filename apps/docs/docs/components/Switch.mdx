## Installation

```bash
npx the-dig@latest add Switch
```

## Props

| Prop            | Type                                                                          | Default   |
| --------------- | ----------------------------------------------------------------------------- | --------- |
| `color`         | `"brand"` \| `"info"` \| `"success"` \| `"warning"` \| `"danger"` \| `"gray"` | `"brand"` |
| `size`          | `"sm"` \| `"md"` \| `"lg"`                                                    | `"md"`    |
| `...props`      | `DetailedHTMLProps<InputHTMLAttributes<HTMLInputElement>, HTMLInputElement>`  | -         |
| `className`     | `string`                                                                      | -         |
