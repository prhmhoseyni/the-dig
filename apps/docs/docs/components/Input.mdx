## Props

| Prop             | Type                                    | Default                                                                       |
| ---------------- | --------------------------------------- | ----------------------------------------------------------------------------- |
| `variant`        | `"primary"` \| `"secondary"`            | `"primary"`                                                                   |
| `hasError`       | `boolean`                               | `false`                                                                       |
| `startAdornment` | `ReactNode`                             | -                                                                             |
| `endAdornment`   | `ReactNode`                             | -                                                                             |
| Other props      | `InputHTMLAttributes<HTMLInputElement>` | [Document](https://developer.mozilla.org/en-US/docs/Web/API/HTMLInputElement) |
