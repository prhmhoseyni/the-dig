## Code

```tsx

  <SegmentControl.Divider />
  <SegmentControl.Item value="2">عنوان نمونه 2</SegmentControl.Item>
```

## Props

| Prop       | Type                       | Default |
| ---------- | -------------------------- | ------- |
| `value`    | `string`                   | -       |
| `onChange` | `(value: string) => void`  | -       |
| `size`     | `"sm"` \| `"md"` \| `"lg"` | `"md"`  |
| `children` | `ReactNode`                | -       |
