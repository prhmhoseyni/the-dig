# Tooltip

## Code

```tsx
import Tooltip from "@repo/ui/Tooltip";

<Tooltip id="top" content="این یک راهنمای ابزار است." place="top">
  <button>top</button>
</Tooltip>;
````

## Installation

```bash
npx the-dig@latest add Tooltip
```

## Props

| Prop       | Type                                                                                                                                                                                       | Default |
| ---------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ | ------- |
| `id`       | `string`                                                                                                                                                                                   | -       |
| `content`  | `string`                                                                                                                                                                                   | -       |
| `place`    | `"top"` \| `"bottom"` \| `"left"` \| `"right"` \| `"left-start"` \| `"left-end"` \| `"right-start"` \| `"right-end"` \| `"top-start"` \| `"top-end"` \| `"bottom-start"` \| `"bottom-end"` | `"top"` |
| `children` | `ReactElement`                                                                                                                                                                             | -       |
