## Props

| Prop             | Type                                      | Default                                                                                                         |
| ---------------- | ----------------------------------------- | --------------------------------------------------------------------------------------------------------------- |
| `variant`        | `"primary"` \| `"secondary"`              | `"primary"`                                                                                                     |
| `hasError`       | `boolean`                                 | `false`                                                                                                         |
| `startAdornment` | `ReactNode`                               | -                                                                                                               |
| Others           | `SelectHTMLAttributes<HTMLSelectElement>` | <RedirectLink href="https://developer.mozilla.org/en-US/docs/Web/API/HTMLSelectElement">Document</RedirectLink> |
