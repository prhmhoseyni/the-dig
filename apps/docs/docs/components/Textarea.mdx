```tsx

````

## Installation

```bash
npx the-dig@latest add Textarea
```

## Props

| Prop             | Type                                                                                  | Default     |
| ---------------- | ------------------------------------------------------------------------------------- | ----------- |
| `variant`        | `"primary"` \| `"secondary"`                                                          | `"primary"` |
| `hasError`       | `boolean`                                                                             | `false`     |
| `startAdornment` | `ReactNode`                                                                           | -           |
| `className`      | `string`                                                                              | -           |
| `...props`       | `DetailedHTMLProps<TextareaHTMLAttributes<HTMLTextAreaElement>, HTMLTextAreaElement>` | -           |
