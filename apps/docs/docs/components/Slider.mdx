## Code

```tsx
import Slider from "@repo/ui/Slider";
import { useState } from "react";

const [singleValue, setSingleValue] = useState(20);
const [rangeValue, setRangeValue] = useState<[number, number]>([30, 70]);

<Slider value={singleValue} onChange={(v) => setSingleValue(v as number)} />

<Slider
  range
  value={rangeValue}
  onChange={(v) => setRangeValue(v as [number, number])}
/>
```

## Installation

```bash
npx the-dig@latest add Slider
```

## Props

| Prop                    | Type                                                              | Default                      |   |
| ----------------------- | ----------------------------------------------------------------- | ---------------------------- | - |
| `value`                 | `number` \| `[number, number]`                                    | -                            |   |
| `onChange`              | \`(value: number                                                  | \[number, number]) => void\` | - |
| `range`                 | `boolean`                                                         | `false`                      |   |
| Other `rc-slider` props | See [`rc-slider` docs](https://github.com/react-component/slider) | -                            |   |
